# Directus Local CMS

Folder ini menyimpan catatan konfigurasi Directus untuk website YPK Elkana.

Untuk simulasi lokal, jalankan dari root project:

```bash
docker compose up -d
```

Directus akan tersedia di `http://127.0.0.1:8055`.

Schema dan permission dibuat melalui Directus admin UI mengikuti:

- `docs/DIRECTUS_SCHEMA.md`
- `docs/DIRECTUS_PERMISSIONS.md`
