# Struktur Pages

Folder ini berisi halaman route website.

- `public`: halaman utama yayasan seperti beranda, tentang, berita umum, PPDB, kontak, dan galeri.
- `units`: halaman khusus unit pendidikan seperti TK, SD, SMP, serta berita per unit.
- `shared`: halaman umum yang dipakai lintas route, misalnya halaman 404.

Kalau hanya mengubah isi teks atau data, edit folder `src/content`, bukan file di folder ini.
