# Panduan Edit Konten Website

Folder ini dibuat supaya pihak sekolah bisa mengubah isi website tanpa menyentuh komponen React.

Edit file sesuai kebutuhan:

- `schoolProfile.js`: nama yayasan, alamat, kontak, jam layanan, teks utama beranda.
- `visionMission.js`: visi, misi, nilai sekolah.
- `navigation.js`: menu utama website.
- `units/tk.js`: konten khusus TK.
- `units/sd.js`: konten khusus SD.
- `units/smp.js`: konten khusus SMP.
- `teachers/teachers.js`: data guru dan staf.
- `admissions.js`: alur PPDB.
- `news.js`: berita dan agenda.
- `gallery.js`: foto galeri.

Tips edit:

- Jangan ubah nama variable seperti `school`, `navItems`, atau `teachers`.
- Ubah hanya teks di dalam tanda kutip.
- Untuk menambah data, copy format object yang sudah ada lalu ganti isinya.
- Simpan file, lalu refresh website.
