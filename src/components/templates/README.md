# Template Components

Folder ini berisi komponen section yang menyusun halaman, misalnya section berita, guru, program, fasilitas, dan profil.

Komponen di sini bertugas mengatur layout dan tampilan. Data yang ditampilkan sebaiknya tetap diambil dari `src/content`, supaya pihak sekolah bisa update konten tanpa mengubah kode template.
